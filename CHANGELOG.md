# Changelog

## 1.0.5

- Aligned the run/debug console background with the editor canvas.
- Moved block-terminal backgrounds and selected blocks into the same blue-black
  palette without changing output text colors.

## 1.0.4

- Added a blue-black UI palette with distinct editor, tool-window, toolbar, and
  raised-surface levels.
- Reduced the contrast of UI borders, method separators, right-margin guides,
  and indent guides.
- Preserved the Catppuccin Mocha syntax foreground colors.
